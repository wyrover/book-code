StlClrLibrary 示例：演示 STL/CLR 功能

本示例演示一些可在使用 STL/CLR 库时使用的功能。

示例将实现一个简单的冒泡排序算法。将显示对话框，以便直观地演示排序过程，并允许用户逐句通过排序。

示例将此算法生成到名为 StlClrLibray 的库中。此库同时包含托管代码和本机代码，并且可以通过托管代码调用算法，或者将其封送到算法的本机代码版本。

生成并运行此示例

  1. 在 Visual Studio 开发环境中打开解决方案文件 StlClr Sample.sln。

  2. 在“生成”菜单中，单击“生成解决方案”。

  3. 在“调试”菜单上，单击“开始执行(不调试)”。
