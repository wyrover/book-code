MSMoneyDemo 示例：Microsoft Money 用户界面示例

MSMoneyDemo 示例应用程序演示如何使用 MFC 创建与 Microsoft Money 类似的用户界面。

安全说明：

提供此代码示例是为了阐释一个概念，它并不代表最安全的编码实践，因此不应在应用程序或网站中使用此代码示例。 对于超出本示例代码的预期用途以外的使用所造成的偶然或继发性损失，Microsoft 不承担任何责任。

获取示例和安装示例的说明：

  • 在 Visual Studio“帮助”菜单上单击“示例”。

    有关更多信息，请参见定位示例文件。

  • 示例的最新版本和完整列表可以从 Visual Studio 2008 Samples（Visual Studio 2008 示例）网页获取。

  • 还可以在计算机的硬盘上查找示例。 默认情况下，会将示例和自述文件复制到 \Program Files\Visual Studio 10.0\Samples\ 下的文件夹中。 对于 Visual Studio 学习版而言，所有示例都在网上提供。

生成并运行示例

生成并运行 MSMoneyDemo 示例

  1. 打开解决方案 MSMoneyDemo.sln。

  2. 选择适当的解决方案平台和配置。

  3. 在“生成”菜单中单击“生成”。

  4. 在“调试”菜单上单击“开始调试”或“开始执行(不调试)”。
